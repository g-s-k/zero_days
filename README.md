# zero_days: a condescending message

Example usage and output:

```shell
$ zero_days "we last had this conversation"

+------------------------------------------------------------------------------+
|                                                                              |
|                                 It has been                                  |
|                                                                              |
|                                      ____                                    |
|                                     / __ \                                   |
|                                    / / / /                                   |
|                                   / /_/ /                                    |
|                                   \____/                                     |
|                                                                              |
|                                     days                                     |
|                                                                              |
|                     since we last had this conversation                      |
|                                                                              |
+------------------------------------------------------------------------------+

```

### install

```shell
$ cargo install zero_days
```
